# bmi-calculator

A console application for calculating body mass index.

## Prerequisites

 * Git
 * A C++ compiler
 * CMake

## Installation

To download, build, install, and run:

    git clone https://github.com/cruzanta/bmi-calculator.git
    cd bmi-calculator
    cmake .
    make
    ./bin/BMIcalculator
